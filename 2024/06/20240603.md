## [1732B. Ugu](https://codeforces.com/problemset/problem/1732/B)

```
for _ in range(int(input())):
    n = int(input())
    s = input()

    count = 0
    i = 0
    while i < n - 1 and s[i] == "0":
        i += 1

    while i < n - 1:
        if s[i] != s[i + 1]:
            count += 1
        i += 1

    print(count)
```
